This folder contains the output datasets in CSV format generated by the implemented models.

## Content Description:
- **network_CE0_pydhn_results.csv**: Results from the steady-state simulation implemented with pydhn (CE0 pydhn).
- **network_CE1_pydhn_results.csv**: Results from the dynamic simulation implemented with pydhn  (CE1 pydhn).
- **network_CE0_pandapipes_results.csv**: Results from the steady-state simulation implemented with pandapipes (CE0 pandapipes).
- **network_CE1_pindepipes_results.csv**: Results from the dynamic simulation implemented with pandapipes (CE1 pandapipes).
- destest_network_0_results.csv: Results from the validated models throught the DESTEST (CE0).
- destest_network_1_results.csv: Results from the validated models throught the DESTEST (CE1 modelica).

## Important Notes:
- **Automatic Updates**: These files are automatically generated or overwritten every time the Python scripts (`Network_0_pyDHN.ipynb` or `Network_1_pyDHN.ipynb`) are executed.
- **Data Preservation**: If you wish to save a specific simulation run, please rename the file or move it to a different subfolder before running the script again.
